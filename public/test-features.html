AI model proxy service that forwards requests to OpenRouter API with custom model mapping.

POST /openrouter/chat/completions

🔐 Authentication & Security

JWT authentication, API key guards, rate limiting, and security middleware.

API Key: Bearer token required
Rate limiting: 60-10000 requests/min

PostgreSQL with TypeORM, Redis caching, and connection pooling.

Database: PostgreSQL (Supabase)
Cache: Redis (cache-manager)
Queue: Bull (Redis-based)

⚡ Performance Features

HTTP caching interceptor, request throttling, and async processing.

Cache TTL: Configurable

Interceptors: Response formatting

WebSocket support, worker processes, and modular architecture.

WebSocket Module: Available

Queue Module: Task processing
